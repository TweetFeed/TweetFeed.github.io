API

TweetFeed.live API (limited to 20k results per search)

Introduction

Open and Free API to get IOCs shared by the community at Twitter.

Endpoint 
GET https://api.tweetfeed.live/{time}/{filter1}/{filter2}

https://api.tweetfeed.live/today/

Key Required? Possible values
time Required

today (Today starting 00:00 UTC)

week (Last 7 days)

month (Last 30 days)

year (Last 365 days)
filter1 Optional

User (@malwrhunterteam / @1ZRR4H / @BushidoToken ...)

Type (url / domain / ip / sha256 / md5)

Tag (phishing / ransomware / CobaltStrike ...)
filter2 Optional

User (@malwrhunterteam / @1ZRR4H / @BushidoToken ...)
